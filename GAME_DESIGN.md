# BudLords - Addictive Game Loop Design

## Core Philosophy
**"Just one more harvest, one more sale, one more quest..."**

The game is designed to create multiple overlapping short-term and long-term goals that keep players engaged through:
- Immediate rewards (harvests, sales)
- Short-term goals (daily quests, timed events)
- Medium-term goals (skill unlocks, achievements, streaks)
- Long-term goals (prestige, legendary strains, mastery)

## Engagement Systems

### 1. Daily Quest System üéØ
**Purpose:** Give players clear daily objectives

**Features:**
- 3 randomized daily quests reset at midnight
- Varied quest types (harvest, sell, earn, smoke, crossbreed)
- Progress bars with visual feedback
- Instant gratification on completion
- Bonus reward for completing all 3 quests

**Rewards:**
- Money: $500-1,500 per quest
- XP: 50-150 per quest
- Completion bonus: $5,000 + 500 XP + 2x earnings for 30 minutes

**Psychology:**
- Checklist satisfaction
- FOMO (don't want to miss daily rewards)
- Variety prevents monotony
- Clear achievable goals

### 2. Login Streak System üî•
**Purpose:** Encourage daily engagement

**Rewards:**

- Monthly (30 days): Special legendary seed

**Psychology:**
- Loss aversion (don't want to break streak)
- Commitment and consistency
- Social proof (compare streaks with friends)
- Habit formation

## Retention Mechanics

### Short-Term (Session Length: 30-120 minutes)
1. Daily quests visible progress
2. Timed events create peaks
3. Harvest cycles (5-10 minutes)
4. Sale opportunities
5. Immediate visual feedback

### Medium-Term (Daily Return: 7-30 days)
1. Daily quest reset
2. Login streak maintenance
3. Skill point accumulation
4. Buyer relationship building
5. Achievement unlocks

### Long-Term (Months)
1. Complete achievement collection
2. Multiple prestige levels
3. Legendary strain collection
4. Top leaderboard positions
5. Mastery of all systems

## Final Notes

The key to an addictive game loop is:
1. **Clear goals** - Always know what to do next
2. **Visible progress** - See improvements constantly
3. **Variety** - Multiple ways to play and progress
4. **Rewards** - Frequent positive reinforcement
5. **Social** - Competitive and cooperative elements
6. **Mastery** - Depth for dedicated players
7. **Surprise** - Random events and discoveries

Every system should answer: "What keeps the player playing for just 5 more minutes?"
